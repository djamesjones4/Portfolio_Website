Doctor's Note

Doctor's Note is a native iOS application to help healthcare professionals connect with patients by sharing session notes.

Technologies
  • chevron_rightSwift 3
    Swift 3 was used to create the entire client side mobile application, including JRON and REST services.
  • chevron_rightXcode
    Xcode is the development environment used to create this app. It was used for implementation of Swift as well as the visual components of the graphical interface.
  • chevron_rightNode.js
    Node.js is the server-side technologie used to host all stored user information and perform authentication and authorization.
  • chevron_rightPostgreSQL
    PostgreSQL is the database used to store all user information.
