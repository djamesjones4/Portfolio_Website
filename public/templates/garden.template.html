Garden Growers

Garden Growers is a full stack RESTful web application for connecting gardeners in a social network.


https://github.com/djamesjones4/gardengrowers
Technologies
  • chevron_rightGoogle OAuth 2.0
    Google's OAuth 2.0 API was used for third party authentication.
  • chevron_rightCharts.js
    Charts.js API was used to create the dynamic graphical interfaces for users dashboards.
  • chevron_rightHTML
    HTML is used to create all of the content in this application.
  • chevron_rightJavascript
    Javascript is the language used for both serverside and clientside coding.
  • chevron_rightJQuery
    JQuery is the client side Javascript library used for DOM manipulation and dynamic generation.
  • chevron_rightCSS
    CSS along with Charts.js and svg graphics are used for the display of this application.
  • chevron_rightNode.js
    Node.js is the server-side technology used to manage the RESTful routing of this application.
  • chevron_rightExpress.js
    Express.js is the server-side framework used to manage the RESTful routing of this application.
  • chevron_rightPostgreSQL
    PostgerSQL is the database used in this application.
  • chevron_rightTwitter Bootstrap
    Twitter Bootstrap is the CSS framework used in this application.
