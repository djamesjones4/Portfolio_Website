Yoga Buddy - https://github.com/djamesjones4/YogaBuddy

Yoga Buddy is a full stack RESTful web application allowing Yoga practitioners to get advice on they posture from certified yoga professionals and the larger yoga community. Ita allows users to upload images and video for commenting, as well as participate in live chats about yoga.

Technologies
  • chevron_rightAngularJS
    AngularJs is used as a client side MVC framework to make this a single-page web application.
  • chevron_rightJavascript
    Javascript is the language used for both serverside and clientside coding.
  • chevron_rightCSS
    CSS along with svg graphics and AngularJS are used for the display of this application.
  • chevron_rightNode.js
    Node.js is the server-side technology used to manage the RESTful routing of this application.
  • chevron_rightExpress.js
    Express.js is the server-side framework used to manage the RESTful routing of this application.
  • chevron_rightPostgreSQL
    PostgerSQL is the database used in this application.
  • chevron_rightTwitter Bootstrap
