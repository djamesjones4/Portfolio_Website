(dot)HasPrettyColors

(dot)HasPrettyColors is a front-end web application to assist web developers in creating color schemes for sites.


https://github.com/djamesjones4/-dot-Has-Pretty-Colors
Technologies
  • chevron_rightColour Lovers API
    the Colour Lovers API was used to integrate searchable color palettes.
  • chevron_rightHTML
    HTML is used to create all of the content in this application.
  • chevron_rightJavascript
    Javascript is the language used for both serverside and clientside coding.
  • chevron_rightJQuery
    JQuery is the client side Javascript library used for DOM manipulation and dynamic generation.
  • chevron_rightCSS
    CSS along with Charts.js and svg graphics are used for the display of this application.
