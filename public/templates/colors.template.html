(dot)HasPrettyColors

(dot)HasPrettyColors is a front-end web application to assist web developers in creating color schemes for sites. It allows the creation of custom color schemes and allows the developer to apply them to a generic website template.

*A full-stack version of (dot)HasPrettyColors utilizing React and C# is currently in the works.


https://github.com/djamesjones4/-dot-Has-Pretty-Colors
Technologies
  • chevron_rightColour Lovers API
    The Colour Lovers third party API was integrated into the front-end to include searchable color palettes.
  • chevron_rightHTML
    HTML is used to create all of the static content in this application.
  • chevron_rightJavaScript
    JavaScript is the language used for all dynamic aspect of the website.
  • chevron_rightJQuery
    JQuery is a client side JavaScript library used for DOM manipulation and dynamic generation.
  • chevron_rightCSS
    CSS along with Twitter's Bootstrap CSS Framework are the language and framework used for the styling of this application.
